### 2.2 资产状态分类

| 状态 | 图标 | 定义 |
|------|------|------|
| 已受影响 | 🔴 | 已确认停产、停运、减产、降负、不可抗力、交付下降、出口暂停、受损或关停退出 |
| 间接受限 | 🟡 | 存在外部约束或传导风险，尚未确认该资产自身减产/停运 |
| 未确认直接受影响 | 🟢 | 公开信息明确显示未受损、运营正常、已恢复或影响为0 |
| 情况不明 | ⚪ | 有风险信号但不能确认受影响也不能确认未受影响 |
| 物理损毁 | 🔴 | 设施遭直接物理破坏 |
| 物流性停产 | 🔵 | 因物流/出口链路中断导致库满停产 |
| 永久关闭 | ✖️ | 产能永久退出 |

### 2.3 地缘传导链节点

```
1. 地缘冲突升级
   ↓
2A. 海峡关闭/绕行有限    2B. 油田/炼厂/裂解装置受损
   ↘↙
3. 供应收缩与运输受阻并行
   ↓
4. 原油价格上行压力
   ↓
5A. 库存缓冲有限    5B. 中短期供应边际减少    5C. 中长期供给收缩
   ↓
6. 能化产品价格与盈利弹性抬升
```

## 七、边际变化判定规则

当搜索到新的炼化装置新闻时，按以下规则判定是否为边际变化：

1. **状态变更**: 资产从"正常/间接"变为"已受影响"，或从"已受影响"变为"永久关闭/复产"
2. **产能损失数值变化**: 与基线表中的数值不一致（新减产/复产）
3. **修复ETA变化**: 修复进度提前或推迟
4. **新增资产**: 基线表中没有的新受影响资产
5. **海峡通行状态变化**: 从限制通行→关闭，或从关闭→恢复
6. **库存数据更新**: 与上期相比出现显著变化（>5%变动需标注）
7. **新的不可抗力声明**: 运营商发布新的FM通知
8. **地缘事件升级/降级**: 新的袭击、停火、制裁等
