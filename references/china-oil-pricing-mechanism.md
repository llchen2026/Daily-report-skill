# 中国成品油定价机制 — 法规核心内置

> 法规依据：《石油价格管理办法》（发改价格〔2016〕64号，2016年1月13日起实施）
> 主管机关：国家发展和改革委员会（NDRC）

---

## 一、定价公式（核心）

中国汽柴油**零售价/批发价**实行**政府指导价**，按以下公式每10个工作日机械调整：

```
调价幅度（元/吨） = 基准价格 × 变化率

变化率 = (本期10工作日移动均价 - 上期10工作日移动均价) / 上期10工作日移动均价
```

### 移动均价基准

| 原油品种 | 产地 | 定位 |
|---------|------|------|
| Brent（布伦特） | 北海 | 欧洲基准 |
| Dubai（迪拜） | 中东 | 亚洲基准 |
| Minas（米纳斯） | 印度尼西亚 | 亚太基准 |

三者按发改委内部权重加权（公开文件未明确权重比例，市场普遍估算 Brent: Dubai: Minas ≈ 4:3:3 至 3:4:3）。

---

## 二、调价规则

| 要素 | 规则 |
|------|------|
| **调价周期** | 每 **10个工作日** 调整一次（约每月2次） |
| **生效时间** | 调价发布当日 **24时** 生效 |
| **调价门槛** | 变动幅度 < **50元/吨** 时搁置（不作调整） |
| **搁置处理** | 未调金额纳入下次调价时 **累加或冲抵** |
| **发布方式** | 发改委以 **信息稿** 形式发布（不再发正式调价文件） |

---

## 三、地板价 / 天花板价（管控区间）

```
＜ 40美元/桶     → 地板价生效：不再下调，差额纳入风险准备金（用于节能减排/油品升级/能源安全）
40～130美元/桶   → 正常按公式调价
＞ 130美元/桶    → 天花板价生效：原则上不调或少调
```

### 风险准备金

- 触发条件：国际油价 < 40美元/桶时
- 来源：成品油未调减金额
- 用途：节能减排、提升油品质量、保障石油供应安全
- 管理：专项账户存储，经国家批准后使用

---

## 四、执行层级

| 价格类型 | 定价方式 | 说明 |
|---------|---------|------|
| **原油价格** | 市场调节价 | 随行就市 |
| **汽柴油零售价** | 政府指导价 | 最高零售价由发改委制定 |
| **汽柴油批发价** | 政府指导价 | 最高批发价按零售价扣减流通差价 |
| **专项用户供应价** | 政府指导价 | 向铁路/交通等专项用户 |
| **储备供应价** | 政府定价 | 向国家储备/兵团 |
| **液化石油气** | 市场调节价 | 2016年放开，供需协商 |

### 最高零售价格构成

```
最高零售价 = 国际原油成本 + 国内平均加工成本 + 税金 + 合理流通费用 + 适当利润
```

## 五、对投资观点的影响（分析指引）

### ✅ 正确逻辑

中国汽柴油价格是**公式化联动**的，核心传导路径：

```
Brent/Dubai/Minas 十日移动均价变化率
    → 调价幅度 ≥ 50元/吨？
        → 是：每10工作日调价（24时生效）
        → 否：搁置累加
    → 是否触发管控区间？
        → <40美元：地板价保护
        → >130美元：天花板价
```

**因此**：中国汽柴油价格的边际变化 = **国际原油移动均价的边际变化**，而非"政策预期"或"调控意图"。

### ❌ 常见错误

以下表述在分析中**禁止使用**：

| 错误表述 | 问题 |
|---------|------|
| "中国成品油调价上涨预期" | 定价是公式化联动，不存在主观预期 |
| "政策面推动汽柴油涨价" | 正常区间内无政策干预空间 |
| "发改委可能调高油价" | 非 discretionary decision，是机械执行公式 |
| "调价预期支撑市场情绪" | 调价窗口是公开日历，不产生预期差 |

### ✅ 正确表述示例

| 维度 | 正确分析方向 |
|------|-------------|
| **基本面** | 十日移动均价当前变化率 → 是否突破50元/吨门槛 → 下一窗口调价方向 |
| **成本链路** | Brent+Dubai+Minas三篮子移动均价 → 中国汽柴油调价幅度 → 国内FU/LU/PG期货定价锚 |
| **政策面** | 仅在接近40/130管控区间边界时才相关（地板价保护/天花板价压制） |
| **情绪面** | 调价窗口前夜的市场博弈（但不构成趋势性预期差） |
